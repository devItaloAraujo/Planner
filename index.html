? Como Funciona o Sistema de Blocos

⏰ Sistema de 30 Minutos

Todos os blocos são organizados em slots de 30 minutos. A duração mínima é 30 minutos.

📝 Atividades Curtas

Para atividades menores que 30 minutos, você pode:

  • • Combinar 2 atividades no mesmo bloco
  • • Usar o campo "Nome e Comentários" para detalhar

🎯 Dicas Importantes

  • • Durações: 30m, 1h, 1h30m, 2h, etc.
  • • Arraste blocos para reposicionar
  • • Clique em um bloco para editar
  • • Blocos colocados na grade podem começar em :00 ou :30

Memory Card

? Como Funciona o Backup

💾 Onde Ficam Salvos os Dados

Seus dados ficam salvos no seu navegador (Chrome, Firefox, etc.) em um local chamado "Local Storage". É como um arquivo temporário que fica no seu computador ou celular.

⚠️ Quando os Dados Podem Ser Perdidos

  • • Limpar dados do navegador
  • • Usar modo incógnito/privado
  • • Desinstalar o navegador
  • • Formatar o computador/celular
  • • Usar outro navegador

🛡️ Como se Proteger

  • • Use o "Exportar - Copiar" regularmente
  • • Cole o texto em um arquivo de texto
  • • Salve em um local seguro (Google Drive, etc.)
  • • Faça backup antes de limpar o navegador
